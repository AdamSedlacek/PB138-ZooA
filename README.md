Zadání projektu:
Rozšíření mobilní aplikace pro Android a iOS. Implementováno v React Native.

Návod k instalaci:<br/>
npm install<br/>
npm install react-native-sound --save<br/>
npm install -g react-native-cli<br/>
react-native run-android<br/>
